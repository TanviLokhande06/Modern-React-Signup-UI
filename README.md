# 🚀 Modern React Signup UI

A clean and responsive **User Registration Form** built with React.
This project demonstrates form handling, validation, and modern UI design using Tailwind CSS.

---

## ✨ Features

* 🔐 Strong form validation
* 📧 Email format validation
* 🔑 Password rules enforcement:

  * Minimum 8 characters
  * At least 1 uppercase letter
  * At least 1 special character
* 🔁 Confirm password matching
* 🚫 Prevent duplicate email registration
* ⚡ Instant error feedback
* 🔔 Toast notifications using React Toastify
* 🎨 Modern and responsive UI (Tailwind CSS)
* 👤 User list display with card-style layout

---

## 🛠️ Tech Stack

* **React** (Functional Components + Hooks)
* **Tailwind CSS**
* **React Toastify**

## 🧠 Validation Rules

* Email must follow a valid format
* Password must:

  * Be at least 8 characters long
  * Contain at least one uppercase letter
  * Contain at least one special character
* Confirm Password must match Password
* Duplicate emails are not allowed

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/modern-react-signup-ui.git
```

### 2. Navigate into the project

```bash
cd SignUp
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

---

## 📦 Dependencies

Install required package:

```bash
npm install react-toastify
```
